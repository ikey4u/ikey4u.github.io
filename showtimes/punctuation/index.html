录入音标的时候, 只有一些不常见的字符难以录入, 为了方便录入, 使用下面的映射规则: 

i: → /iː/
E  → /æ/
3: → 3:
@  → /ə/
^  → /ʌ/
U  → /ʊ/
): → /ɔː/
D  → /ɒ/
A  → /ɑː/
S  → /ʃ/
0  → /θ/
R  → /ʒ/
Z  → /ð/
