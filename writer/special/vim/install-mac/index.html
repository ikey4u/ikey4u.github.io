配置完成后执行如下命令完成安装

安装 xcode

xcode-select --install
sudo xcode-select -switch /Library/Developer/CommandLineTools

查看 vim 信息

brew search macvim
brew info macvim

安装 macvim

brew install cmake
brew install macvim --with-lua --with-override-system-vim
