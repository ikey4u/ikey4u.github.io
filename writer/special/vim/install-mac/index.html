编译环境参考 ./install-overview.txt 或 http://ahageek.com/writer/specia...

从这里 https://github.com/macvim-dev/ma... 下载二进制文件,安装完成后, 输入 :version 命令确认 python 为动态加载模式,也就是包含以下输出 +python/dyn +python3/dyn, 然后设置环境变量

export PATH="/Applications/MacVim.app/Contents/bin":$PATH

由于 YCM 依赖于 cmake , 所以安装一下 cmake

brew install cmake

克隆 vim 配置文件到 ~/.vim 并设置 user.vim 中的 python 路径.
