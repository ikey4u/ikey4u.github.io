vim 自定义选项
by bugnofree
Publish → 2019-01-17 Update → 2019-09-02

安装完成后, 需要在  $HOME/.vim  下新建一个 user.vim 文件, 在该文件中配置几个选项,以完成自定义, 可以复制 .vim 目录下的 _user.vim 为 user.vim, 然后修改 user.vim 即可.

_user.vim 中各个选项的含义如下

  • g:pyhome

    是 python 的主目录, 可以通过如下命令获得

    python3 -c "import sys; print(sys.executable)"

    比如它输出

    C:\Users\Administrator\AppData\Local\Programs\Python\Python37\python.exe

    那么 python 主目录就是

    C:\Users\Administrator\AppData\Local\Programs\Python\Python37

    然而在 windows 下你需要替换反斜线为斜线

    C:/Users/Administrator/AppData/Local/Programs/Python/Python37

    windows 用户也是一样的.

  • g:pydll

    python3 动态链接库的绝对路径. windows 用户必须设置.

  • g:terminal
    • windows  C:\Windows\system32\cmd.exe 
    • linux 自己找吧
    • mac 参考默认设置
  • g:openex
    • windows  start 
    • linux 自己找吧
    • mac  open 
  • g:myplan, g:mybook, g:myidea, g:year

    相关文档快速打开, 如果你需要则配置, 否则可忽略.

  •  idapro ,  idaprotag 

    如果你不知道是啥, 那么你不用管.
