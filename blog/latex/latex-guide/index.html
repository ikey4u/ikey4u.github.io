环境

安装一些必要的中文字体.打开文件 /etc/fonts/fonts.conf, 里面是自定义的一些字体目录,比如说 ~/.fonts,我们可以将下载好的字体放到这个目录, 然后执行下面的命令: 

 sudo mkfontscale && sudo mkfontdir && sudo fc-cache -fv

我们可以使用命令 fc-list :lang=zh-cn看看系统都安装了哪些字体, 注意上述命令中 fc-list 和 冒号(:)之间是有一个空格的.由于输出的每一行包含了字体所支持的样式, 这个我们不需要过多关注,因此可以使用命令 fc-list :lang=zh-cn | cut -f1,2 -d":"来输出前两个字段, 我的电脑上输出如下:

但我们在 latex 中使用字体时,所用的字体名称就是每一行第一个冒号之后所显示出来的名称, 有的字体有多个名称,那么它们会用逗号来间隔,我们随便选择哪一个都可以.举个例子, 如幼圆字体,你既可以用 幼圆,也可以用 YouYuan.

所有的 latex 文档都是以 documentclass 开始的,同时我们大多数时间用的是中文书写, 所以首先需要解决中文环境问题,要解决中文问题,第一是需要让 latex 支持中文,第二需要设置中文字体. 一个较小的中文环境如下:

\documentclass[a4paper,11pt]{article}
%中文自动换行
\XeTeXlinebreaklocale "zh"
\XeTeXlinebreakskip = 0pt plus 1pt
%中文编码支持
\usepackage{xltxtra,fontspec,xunicode}
%中文支持包
\usepackage{xeCJK}
%书签
\usepackage{CJKutf8}
%中文字体
%\setCJKmainfont{YouYuan}
\setCJKsansfont{SimSun}
%自定义要使用的中文字体
\setCJKmonofont{YaHei Consolas Hybrid}
\setCJKfamilyfont{SimSun}{SimSun}
\setCJKfamilyfont{SimHei}{SimHei}
\setCJKfamilyfont{FangSong}{FangSong}
\setCJKfamilyfont{KaiTi}{KaiTi}
\setCJKfamilyfont{YouYuan}{YouYuan}
\newcommand{\song}{\CJKfamily{SimSun}} % 宋体
\newcommand{\hei}{\CJKfamily{SimHei}}  % 黑体
\newcommand{\fs}{\CJKfamily{FangSong}} % 仿宋
\newcommand{\kai}{\CJKfamily{KaiTi}}   % 楷体
\newcommand{\yy}{\CJKfamily{YouYuan}}  % 楷体
%英文字体
\setmainfont{YouYuan}
%\setmainfont{Segoe Print} %遗憾的是,会导致乱码
\setmonofont{Courier New}
