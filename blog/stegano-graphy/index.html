前言:什么是隐写术

隐写术,英语单词是Steganography,这个单词很长,可以拆为两部分:
第一部分是stegano,希腊语里的意思是"覆盖"
第二部分graphy,英语中的常用词根,表示"写".
那么这两部分连起来就是"将写的覆盖起来",就可以理解为隐写术.

相关软件安装

编译安装stegdetect

该软件的开源地址为:https://github.com/abeluck/stegdetect,下载编译安装的方式如下: 
apt-get install automake1.4
git clone https://github.com/abeluck/stegdetect
cd stegdetect
#如果该地址失效,可切换为:http://www.ahageek.com/blog/stegano-graphy/varied.out
wget -O file/Magdir/varied.out https://raw.githubusercontent.com/file/file/master/magic/Magdir/varied.out
linux32 ./configure
linux32 make
一般的,按照上面的步骤,一般是不会出错的,如果你简单的按照官方上的指导编译,那么你可能遇到的一个问题如下: 
cd . && /root/Downloads/stegdetect/missing aclocal-1.4
WARNING: `aclocal-1.4' is needed, and you do not seem to have it handy on your
         system.  You might have modified some files without having the
         proper tools for further handling them.  Check the `README' file,
         it often tells you about the needed prerequirements for installing
         this package.  You may also peek at any GNU archive site, in case
         some other package would contain this missing `aclocal-1.4' program.
make: *** [aclocal.m4] Error 1
这个说明的automake版本不对,你需要安装automake 1.4版本,你可以使用如下命令查看automake版本: 
dpkg -l | grep automake
#样例输出:ii  automake 1:1.14.1-2ubuntu1  all Tool for generating GNU Standards-compliant Makefiles

输出的信息中,冒号后面断横线之前的数字即为版本号,这里为1.14.1.你也可以在shell中输入automake然后按下tab键, 系统一般会自动补全列出已经安装的版本,使用automake --version会列出默认的automake版本.

上述的编译环境是ubuntu 14.04 x64,其他基于Debian的系统也应该是适应的.但是我没有测试.你可以下载我已经编译好的stegdetect-linux-bin.zip. 对于Windows的版本,你同样可以下载,名字为stegdetect-0.4.zip.
